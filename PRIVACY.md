# Privacy

CourtListener Copy Citation does not collect, store, sell, or transmit user data.

The extension runs only on CourtListener opinion pages. When you select text, it reads the selected text and visible citation metadata from the current page in order to format a citation locally in the browser. It writes the result to your clipboard only when you click one of the copy buttons.

No analytics, tracking, remote code, external API calls, or third-party services are used.
